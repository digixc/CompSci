Welcome to Year 3 Computing at Wellington College International Shanghai

Curriculum Outlines

Unit 0: Office Talks

    In this unit, children will be introduced to using Teams. Learn how to access files, take screen shots and upload work:

  • Teams
  • One note
  • Basic word processing skills

Unit 1: Introduction to Programming

    In this unit, children will be introduced to basic programming skills:

  • Purple Mash Logo Turtle programming
  • Wedo block based coding
  • Code.org block based coding

Unit 2: Research Skills

    In this unit, children will learn how to research topics on the internet and use a search engine safely:

  • Search engines
  • Use of keywords to create effective searches

Unit 3: Presenting Information

    In this unit, children will learn how to present information digitally using Powerpoint focusing on appropriate use of:

  • Colour and contrast
  • Font
  • Style
  • Images

Unit 4: Podcasts / Animations

    In this unit, children will be introduced to creating animations and sharing information via podcasts.

  • Purple Mash
