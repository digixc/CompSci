    In this unit you will learn how to design and build your own web site. The unit will teach you the following skills:

  • What the World Wide Web is and how to use it.
  • Time Management
  • The basics of HTML (Language which is used to create web pages).
  • How to use web design applications ( I have not decided yet).

Your overall assessment during this unit will be based on how well you design and build your very own website(blog). Tasks during this unit will include:

  • .Planning out the website
  • .Using Microsoft Expression to actually create the website including features such as:
  • .Tables (layout)
