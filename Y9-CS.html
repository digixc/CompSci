Options Guide

Unit 0: For a Better Leaner

    An introduction unit for developing students to be a better learner by equiping them with the right skills:

  • Research
  • Time Management
  • Information Synthesis
  • Presentation

Unit 1: Cryptography and Cyber Security

    An unit on cyphers and encryptions and their applications in cyber security:

  • Encryption and Decryption with Caesar Cypher
  • Cyber Security
  • Implementing cyphers using Python
  • PKI and its applications on the Internet

Unit 2: Game Development

    An unit on game programming:

  • Intro to animation and game development
  • How Computer represent colours and shapes
  • Sprites, text and their properties
  • Variables, loops, conditionals and functions
  • Interactive game essentials: inputs, velocity, collision and events

Unit 3: Physical Computing

    An unit on physical computing using BBC Micro:bit, Lego Robotics and Phidgets

  • intro to Micro:bit: hardware components and IDE
  • Input, Output, Sensors and Events
  • Loops, Conditionals and Functions
  • Design and make your own IoT
  • Testing and evaluation

Unit 4: Fundamentals of Computing

    An unit on understanding how computers work and on the applications of computing

  • A Brief History of Computing
  • Key Technologies and Components of a Computer
  • Fetch-Decode-Excution cycle
  • LMC: Little man computer
  • Binary, Hex, Image and Sound
  • IoT, Robotics and AI
