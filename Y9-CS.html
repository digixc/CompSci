Options Guide

Unit 0: The Return of Python (4 weeks)

    Further developing your skills in problem solving using programming language Python, specifically:

  • Develop understanding of data types and purposes of variables
  • Develop understanding the two types of loops and when to use which
  • Develop understanding the using of conditionals and logic operations
  • Develop understanding why modular programming is used

Unit 1: Game Development (9 weeks)

    A unit on game programming using Python pygame module to develop problem solving and programming skills in an engaging way:

  • Intro to animation and game development
  • How computers represent colours and shapes
  • Sprites, text and their properties
  • Variables, loops, conditionals and functions
  • Interactive game essentials: inputs, velocity, collision and events

Unit 2: AI and Machine Learning (4 weeks)

    A unit on the basics of AI and machine learning by introducing some fundamentals concepts in the world of artificial intelligence:

  • What is AI?
  • How machine learning is achieved?
  • Ethics of AI
  • Image recognition
  • Simple chatbot implementation in Python
  • Intelligence, Turing test and singularity

Unit 3: Physical Computing with Rovers (7 weeks)

    A unit on physical computing using Rovers, a programmable 4 wheel vehicle:

  • Intro to Rovers: hardware components and IDE
  • Input, output, sensors and events
  • Loops, conditionals and functions
  • Design and make your own AI-mobiles
  • Testing and evaluation

Unit 4: Cryptography and Cyber Security(4 weeks)

    A unit on cyphers and encryptions and their applications in cyber security:

  • Encryption and decryption with Caesar Cypher
  • Implementing cyphers using Python
  • PKI and cyber security
  • Frequency analysis and computational security

Unit 5: Physical Computing with Phidgets (6 weeks)

    A unit on physical computing using Phidgets and a host of sensors:

  • Intro to Phidgets: hardware components and Programming Interfaces
  • Input, Output, Sensors and Events
  • Loops, Conditionals and Functions
  • Design and make your own IoTs
  • Testing and evaluation
