    An introduction unit reminding students how to access, submit and set up thier work

  • Personalisation
  • Submitting assignments
  • Using OneNote

    An introductory unit on computer science using the BBC Micro: Bit

  • Computational thinking concepts

Unit 2: Python turtle

    A unit introducing students to Python using the Turtle Graphics package

  • Challenges

Unit 3: Algorithms with Flowol

    A unit on algorithms, pseudocode and flowcharst

  • What are algorithms
  • Pseudocode
  • Flowcharts
  • Simulating systems using Flowol

Unit 4: HTML and CSS

    A unit creating a multipage interactive website on a social cause students are interested in

  • Choosing a social cause to support
  • A reminder of HTML from Year 6
  • Creating and linking multiple webpages
  • Creating a quiz
  • HTML and Javascript

Unit 5: App Design Unplugged

    A unit where pupils theorycraft an application and present it as a business pitch

  • Launch and team-building
  • Spotting and selecting problems
  • Creating a good user experience for your users needs
  • Plan the build
  • Present the build

Unit 6: Robotics with Tello Drones

  • Hello Tello
  • Flip the drone
